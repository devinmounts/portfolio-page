Current Conditions is a river gauge site that montiors and presents all water flow data for all U.S. Rivers monitored by the USGS. We accessed the data through their API and parsed and presented the data in scoped and searchable fields. We also utilized Google Fusion Tables in order to generate an interactive prsentation of data & the Foundation framework to ensure it scales for all devices. Check It Out Live!

My first team project with Rails was a refactoring of a previously established e-commerce site, migrating from a WordPress hosted site into a full Rails application. The project included total reformatting of a previous established code base and live application of payment processes and subscription services, among other key components in order to allow for a User to log in an manage their account.

Building the portfolio site provided an opportunity to experiment with HTML, CSS, and Javascript. As well as a chance to gain real world experience with grid systems such as Twitter Bootstrap.

Working throught Michael Hartl's Ruby on Rails Tutorial was a chance to build a full scale Rails application that is a mimic site to Twitter's microblogging platform. Features include User login, subscriptions, the ability to follow/unfollow users and to provide content. All data collected is migrated to a database using SQLite and ActiveRecord.
